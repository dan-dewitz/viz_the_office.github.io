Viewing

Plot designed for a laptop or desktop environment, as legends can be toggled on or off with the click of a mouse to compare different characters.

Plot is responsive to 781px, then only a static png image is displayed. If you are having troubles viewing the plot, maximize the webpage, or switch to laptop or desktop if you are currently on mobile.

Plot can be reset by clicking the 'reset icon' in the top right coner of the plot (only visable on hover).

Data Source

Data is publically available and I first discovered from a Reddit Data is Beautiful Data Viz Battle.
